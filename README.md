# PPT
Утилита, прокладывающая маршрут до хоста.

Использование:
* Скачать репозиторий
* Собрать проект
* Запустить App.exe из консоли с именем хоста или ip 
 Пример:
>  \App\bin\Release\net6.0\App.exe github.com
